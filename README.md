# Домашнее задание к лекции «Decorators»

1. Доработать декоратор logger. Должен получиться декоратор, который записывает в файл main.log дату и время вызова
   функции, имя функции, аргументы, с которыми вызвалась, и возвращаемое значение.
2. Доработать параметризованный декоратор logger. Должен получиться декоратор, который записывает в файл дату и время
   вызова функции, имя функции, аргументы, с которыми вызвалась, и возвращаемое значение. Путь к файлу должен
   передаваться в аргументах декоратора.
3. Применить логгер к приложению с предыдущего д.з. <br>В данном случае я применяю его к "Книге рецептов".
